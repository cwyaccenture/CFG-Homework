Basic Calculator

To sum up two numbers a and b, input SUM(a,b)

To get a divided by b, input DIVIDE(a,b)

TO get a minus b, input SUBTRACT(a,b)

To get a multiplied by b, input MULTIPLY(a,b)

To get a to the power b, input POWER(a,b). Note: Currently, this calculator is really basic. So it can only give out accurate powers for integer b.

To get the square root of a, input SQRT(a)
